% (~4-5 pages)

% Opening: The gut microbiome as neurotransmitter factory
% - 95\% of serotonin produced in the gut by enterochromaffin cells
% - GABA, dopamine precursors, SCFAs all synthesized by gut bacteria
% - The microbiome is not just a bystander — it is a direct participant
%   in neurotransmitter biosynthesis and regulation
% - Yet we lack tools to characterize the community-level structure
%   that supports this production

% The structure problem
% - Individual strains produce neurotransmitters, but production depends on
%   cross-feeding networks: SCFA producers support serotonin-synthesizing taxa,
%   tryptophan metabolizers feed downstream pathways
% - These cross-feeding relationships form a topology — connected components,
%   loops, higher-order structures — that compositional analysis cannot capture
% - Current approaches: strain-by-strain probiotic studies, relative abundance
%   correlations, machine learning on feature vectors
% - What's missing: the SHAPE of the microbial interaction landscape

% Disruption: fungi and antibiotic resistance as topological perturbations
% - Mycobiome (gut fungi): Candida, Aspergillus, mold-derived mycotoxins
%   disrupt bacterial community structure through competition, toxin production,
%   and immune modulation
% - AMR gene carriers: antibiotic-resistant taxa persist through perturbations
%   while susceptible neighbors are eliminated, fragmenting cross-feeding networks
% - Both are external factors that reshape the topology of bacterial communities
%   in ways that compositional metrics miss

% The TDA opportunity
% - Persistent homology captures multi-scale topological features
%   (connected components, loops, voids) in co-occurrence networks
% - Betti curves and persistence landscapes provide vectorized descriptors
%   of community shape, enabling statistical comparison and classification
% - Proven in brain neuroscience but never applied to the microbiome side
%   of the gut-brain axis

% Our contribution (5 novelty claims)
% 1. First application of persistent homology (not Mapper) to bacterial
%    co-occurrence networks — Betti curves and persistence landscapes
%    as microbiome descriptors
% 2. Linking topological features to a functional outcome:
%    neurotransmitter production capacity (serotonin, GABA, dopamine precursors)
% 3. Characterizing mycobiome disruption as topological fragmentation
%    of neurotransmitter-producing community structures
% 4. AMR carriers as topological disruptors: simulating antibiotic perturbation
% 5. Bridging TDA-microbiome and TDA-neuroscience: same topological framework
%    applied to both sides of the gut-brain axis
