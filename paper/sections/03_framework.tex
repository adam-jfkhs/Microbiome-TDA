%   - Ecological meaning: distinct microbial guilds/modules
%   - Healthy gut: few large connected components (integrated community)
%   - Dysbiotic: many small components (fragmented community)
%   - Merging of components during filtration → inter-guild communication
%

%   - Ecological meaning: circular co-occurrence patterns,
%     mutualistic/competitive cycles, metabolic feedback loops
%   - Example: A promotes B, B promotes C, C promotes A
%   - Healthy gut: stable persistent loops (robust feedback mechanisms)
%   - Dysbiotic: loss of loops (broken feedback) or new unstable loops

%   - Ecological meaning: higher-order community organization,
