\section{Introduction}

% (~4-5 pages)

% Opening: The gut microbiome as neurotransmitter factory
% - 95\% of serotonin produced in the gut by enterochromaffin cells
% - GABA, dopamine precursors, SCFAs all synthesized by gut bacteria
% - The microbiome is not just a bystander — it is a direct participant
%   in neurotransmitter biosynthesis and regulation
% - Yet we lack tools to characterize the community-level structure
%   that supports this production

% The structure problem
% - Individual strains produce neurotransmitters, but production depends on
%   cross-feeding networks: SCFA producers support serotonin-synthesizing taxa,
%   tryptophan metabolizers feed downstream pathways
% - These cross-feeding relationships form a topology — connected components,
%   loops, higher-order structures — that compositional analysis cannot capture
% - Current approaches: strain-by-strain probiotic studies, relative abundance
%   correlations, machine learning on feature vectors
% - What's missing: the SHAPE of the microbial interaction landscape

% Disruption: two orthogonal mycobiome axes + AMR
% - Axis 1: Environmental mold exposure (Aspergillus, Penicillium, etc.)
%   as an external perturbation; presence may indicate inhalation/ingestion exposure
% - Axis 2: Endogenous gut mycobiome (Candida, Malassezia) as internal dysbiosis
% - These are orthogonal: a patient can have high environmental exposure with
%   low gut fungal burden, or vice versa
% - AMR gene carriers: antibiotic-resistant taxa persist through perturbations
%   while susceptible neighbors are eliminated, fragmenting cross-feeding networks
% - All three axes reshape bacterial community topology in ways that
%   compositional metrics miss

% The TDA opportunity
% - Persistent homology captures multi-scale topological features
%   (connected components, loops, voids) in co-occurrence networks
% - Betti curves and persistence landscapes provide vectorized descriptors
%   of community shape, enabling statistical comparison
% - Proven in brain neuroscience, protein structure, materials science
%   but not yet applied to microbiome co-occurrence networks

% Biomarker interpretation via evidence-weighted priors
% - We do NOT claim to map taxa to biomarker effects deterministically
% - Instead: evidence-weighted priors linking taxa/modules to inflammatory
%   signaling, used to interpret topological shifts and generate testable
%   hypotheses
% - Each taxon-biomarker edge carries: direction, evidence grade (A/B/C),
%   study types, citations
% - CIRS framework evaluated (not endorsed) as a case study for
%   topology-informed inflammatory signature analysis

% Our contribution (5 novelty claims)
% 1. First application of persistent homology (not Mapper) to bacterial
%    co-occurrence networks with Betti curves and persistence landscapes
%    as microbiome descriptors
% 2. Linking topological features to evidence-weighted inflammatory priors
%    (TGF-beta1, VIP) rather than deterministic biomarker predictions
% 3. Orthogonal decomposition of mycobiome effects: environmental mold
%    exposure proxy vs endogenous gut mycobiome composition
% 4. AMR carriers as topological disruptors: perturbation simulation
% 5. CIRS as a case study for evaluating topology-informed inflammatory
%    signatures in mold-exposed cohorts
